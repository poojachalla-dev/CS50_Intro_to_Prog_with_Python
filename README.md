## 💻 How I Push Code to GitHub

I am using **Git** and **VS Code** to manage and upload my code.

### Steps:

1. Initialize Git (only once)

```
git init
```

2. Add files

```
git add .
```

3. Commit changes

```
git commit -m "Added week 1 problem set"
```

4. Connect to GitHub

```
git remote add origin https://github.com/poojachalla-dev/CS50_Intro_to_Prog_with_Python.git
```

5. Push code

```
git push -u origin main
```
