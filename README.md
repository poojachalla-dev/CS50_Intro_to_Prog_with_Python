## 📘 About the Course

[CS50P](https://cs50.harvard.edu/python/) is a beginner-friendly course by **Harvard University** that teaches programming fundamentals through Python — from variables and loops all the way to object-oriented programming and unit testing.

**What the course covers:**

| Week | Topic |
|------|-------|
| Week 0 | Functions, Variables |
| Week 1 | Conditionals |
| Week 2 | Loops |

| Week 6 | File I/O |
| Week 7 | Regular Expressions |
| Week 8 | Object-Oriented Programming |
| Week 9 | Et Cetera |
